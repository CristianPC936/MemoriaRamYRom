MEMORIA RAM

-----------------------------------------------------------------------

La memoria RAM (Random Access Memory, en español Memoria de Acceso Aleatorio)

es un componente fundamental de un ordenador o dispositivo electrónico que se

utiliza para almacenar temporalmente datos y programas que están siendo utilizados activamente.

A diferencia del almacenamiento permanente, como un disco duro o una unidad de estado sólido (SSD),

la RAM es volátil, lo que significa que pierde su contenido cuando se apaga el dispositivo o se reinicia.
